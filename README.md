# AI Workshop - OpenEdge ABL Project

This project demonstrates OpenEdge ABL (Advanced Business Language) development patterns and best practices for building business applications.

## Project Structure

- **src/**: Source code directory
  - **business/**: Business logic classes and datasets
  - **CustomerWin.w**: Customer management window
  - **ItemWin.w**: Item management window
- **.windsurf/**: Windsurf IDE configuration and workflows
- **doc/**: Documentation files
- **dump/**: Database schema files

## Features

- Business Entity Pattern implementation
- Customer and Item management interfaces
- OpenEdge ABL class-based architecture
- Windsurf IDE integration

## Getting Started

1. Clone this repository
2. Open in Progress Developer Studio or Windsurf IDE
3. Load the Sports2000 database schema from `dump/sports2000.df`
4. Compile and run the application

## Technologies

- OpenEdge ABL (Progress 4GL)
- Business Entity Pattern
- Windsurf IDE workflows
